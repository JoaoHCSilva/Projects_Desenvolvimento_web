Estacionamento do João 🏎️
Bem-vindo ao projeto Estacionamento do João, um sistema de gerenciamento de vagas de estacionamento desenvolvido com tecnologias web front-end. A aplicação permite cadastrar veículos, associá-los a uma vaga, visualizar o status de todas as vagas, editar informações e liberar vagas quando um veículo sai.

Este projeto foi criado como uma forma de aplicar e demonstrar conhecimentos em HTML, CSS e JavaScript puro (Vanilla JS), focando na manipulação do DOM e na criação de uma interface de usuário interativa e responsiva.

🖼️ Prévia do Projeto
✨ Funcionalidades Principais
📝 Cadastro de Veículos: Formulário intuitivo para adicionar novos veículos, registrando nome do cliente, telefone, placa e a vaga a ser ocupada.

🚗 Visualização de Vagas: Um painel visual que mostra todas as vagas do estacionamento, indicando quais estão livres e quais estão ocupadas, com os respectivos dados do veículo.

✏️ Edição de Dados: Possibilidade de clicar em um veículo estacionado e editar suas informações diretamente no formulário.

🗑️ Liberação de Vagas (Exclusão): Funcionalidade para remover um veículo de uma vaga, tornando-a disponível novamente no sistema.

📱 Design Responsivo: A interface se adapta a diferentes tamanhos de tela, garantindo uma boa experiência de uso tanto em desktops quanto em dispositivos móveis.

🚀 Tecnologias Utilizadas
O projeto foi construído utilizando as seguintes tecnologias:

HTML5: Para a estrutura semântica da página.

CSS3: Para a estilização, utilizando conceitos modernos como:

Flexbox e Grid Layout: Para a criação de layouts complexos e alinhamento dos componentes.

Variáveis CSS: Para um gerenciamento de cores e estilos mais fácil e consistente.

Media Queries: Para garantir a responsividade em diferentes dispositivos.

JavaScript (Vanilla JS): Para toda a interatividade, manipulação do DOM e lógica da aplicação, sem o uso de frameworks ou bibliotecas externas.
